# :email: HTML Email - Transactional

This project demonstrates my skills at creating an HTML transactional email.

## Content

Rebuild of an email transaction sent out by Shinola.

This email focuses on a fluid layout that sizes itself using percentage-based
widths to shrink horizontally on narrow screens.

## How to View this Email

Click this [link](html-email-transactional-shinola.netlify.app) to view the
email in a browser.
